pip list                        > lista todas las librerias de python
pip freeze                      > lista todas las librerias instaladas manualmente de python
pip install NOMBRE_LIBRERIA     > instala la libreria y si no la encuentra lanza un error
python -m venv NOMBRE_ENTORNO   > crea un entorno en la ubicacion especificada
source ENTORNO/scripts/activate > levanta el entorno para ser usado usando terminales GIT BASH
source ENTORNO/bin/activate     > levanta el entorno en S.O. Linux
ENTORNO/scripts/activate        > levanta el entorno usando terminales POWERSHELL | CMD
deactivate                      > desactiva el entorno
pip freeze > requirements.txt   > lista las librerias y lo guarda en el archivo requirements.txt
pip install -r requirements.txt > buscar el archivo especificado e instalara todas las librerias declaradas dentro del archivo


PARA USO DE MIGRACIONES
--app NOMBRE_ARCHIVO:VAIABLE_FLASK > OPCIONALMENTE se puede agregar la opcion para indivar donde se ubica el archivo de flask

flask db init  > inicializa nuestro proyecto con las migraciones


flask db migrate -m "Agregue tabla alumnos" > crea una nueva migracion basandose en los cambios que hay en la DB

flask db upgrade > ejecuta las migraciones faltantes en la base de datos
